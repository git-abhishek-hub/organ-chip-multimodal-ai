# organ-chip-multimodal-ai
Multi-modal AI framework for Organ-on-Chip liver toxicity prediction using Cell Painting, transcriptomics, chemical descriptors, and functional biomarkers.

## Objective
Develop a multi-modal AI pipeline for Drug-Induced Liver Injury (DILI) prediction using:
- Cell Painting imaging
- LINCS L1000 transcriptomics
- Chemical descriptors
- Organ-chip biomarkers

## Tech Stack
- Python
- PyTorch
- RDKit
- CellProfiler
- MOFA+
- Scanpy
